A simple and fast weather application that shows the current temperature and weather conditions for any city. Designed with a clean UI, smooth animations, mobile responsiveness, and full dark/light theme support.

**This app uses the [WeatherAPI](https://www.weatherapi.com/) to fetch real-time weather data.**

---

## Features
- Fetches real-time weather data from WeatherAPI.  
- Clean and user-friendly interface.  
- Dark/Light theme toggle (button outside the card).  
- Smooth fade-in animations for results and weather icon.  
- Search using the button or by pressing the Enter key.  
- Fully responsive for mobile and desktop.
